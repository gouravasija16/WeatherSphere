# 🌤️ WeatherSphere
>Real-time weather around the world with AI-powerd advice


![WeatherSphere Preview](preview.png)

## ✨ Features

- 🔍 **City Search** — Search weather for any city worldwide
- 📍 **Geolocation** — Detect your current location automatically
- 🌡️ **Current Weather** — Temperature, condition, feels like, humidity, wind, visibility, pressure
- 🌅 **Sunrise & Sunset** — Accurate local times for any city
- 📅 **5-Day Forecast** — Daily weather predictions with Meteocons icons
- 🤖 **AI Advice** — Gemini AI powered personalized weather tips
- 🎨 **Dynamic Backgrounds** — Background changes based on weather condition
- ♾️ **Recent Searches** — Last 5 searched cities saved locally
- 🌡️ **°C / °F Toggle** — Switch between Celsius and Fahrenheit
- 📱 **Responsive Design** — Works on all screen sizes

## 🛠️ Built With

- HTML5, CSS3, JavaScript (ES6+)
- OpenWeatherMap API
- Google Gemini AI API
- Meteocons Weather Icons
- Font Awesome Icons
- Glassmorphism UI Design

## ⚙️ Setup

1. Clone the repo:
```bash
git clone https://github.com/gouravasija16/WeatherSphere.git
```

2. Create `config.js` in root folder:
```js
const API_KEY = "your_openweathermap_api_key"
const GEMINI_KEY = "your_gemini_api_key"
```

3. Get free API keys:
- OpenWeatherMap → [openweathermap.org](https://openweathermap.org)
- Gemini AI → [aistudio.google.com](https://aistudio.google.com)

4. Open `index.html` with Live Server
